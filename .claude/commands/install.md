Comprehensive Claudio installation system that creates project-specific installations with technology stack-aware documentation and complete workflow integration.

**Installation Types**:
- **Full Installation**: Complete `.claude/` and `.claudio/` directories with project-specific CLAUDE.md
- **Commands-Only**: Streamlined installation with commands and essential agents only

**Project Integration Features**:
- **Technology Stack Detection**: Analyzes project to customize commands and documentation
- **Project-Specific CLAUDE.md**: Creates integration guide tailored to detected technology stack
- **Comprehensive Discovery**: Complete project analysis and capability assessment
- **Workflow Integration**: Sets up phase-based development with current project context

**Installation Components**:
- **Commands**: 10+ workflow commands with project customization
- **Agents**: 23+ specialized agents with extended context integration
- **Extended Context**: 45+ context files organized by category and topic
- **Project Documentation**: Technology-specific guidance and integration patterns

**Usage Patterns**:
- `install ./my-project` - Full installation with project-specific customization
- `install commands` - Commands-only installation for existing setups
- `install` - Install in current directory with auto-detection

Use the install-workflow-agent subagent to orchestrate the complete installation process with project discovery, technology detection, and documentation generation.

**Output**: Complete Claudio installation with project-aware documentation, customized commands, and comprehensive workflow integration ready for immediate use.
