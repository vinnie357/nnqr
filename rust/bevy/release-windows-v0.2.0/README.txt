QUADRADIUS v0.2.0 - Windows Release
=====================================

QUICK START:
1. Double-click quadradius.exe to run
2. Click and drag pieces to move them
3. Press Space to end your turn
4. Collect power orbs by moving over them
5. Use powers during power phase or skip

DEBUG CONTROLS:
- P: Spawn power orb for testing
- O: Show current power inventory  
- I: Generate power test report

FEATURES:
- 12 working power types
- Turn-based strategy gameplay
- Terrain height system
- Automated testing system

For full details see RELEASE_NOTES.md

System Requirements:
- Windows 10+ (64-bit)
- 4GB RAM minimum
- DirectX 11 compatible graphics

No installation required - portable executable!
